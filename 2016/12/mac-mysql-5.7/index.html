远程访问 MySQL

上面 SQL 创建了账号 ‘root’@’%’，% 为 like 通配符，表示任何主机都可以连接。若想创建只能同一个局域网内才能访问，可以写成类似这样，’192.168.2.%’。

MySQL客户端

MySQL 的 GUI 客户端很多[ref]，常见的如：MySQL Workbench（官方免费 wiki）、SQLyog（收费 wiki, github）、HeidiSQL（免费 wiki, github）、Navicat（收费 wiki，支持多平台 Windows/Mac/Linux）。Mac 下比较好的，如：Sequel Pro（免费 github，home，MIT协议）。
