      • ENTRY

        public static final Template ENTRY
        Will provide necessary calls to enable sampling and timing the method or method call.

        Accepts the following tags

        • $TIMED - enables the timing support
        • $SAMPLER=[Const | Adaptive] - selects a sampler, if any
        • $MEAN=<mean> - only when sampling; the mean number of hits between samples
        • $METHODID=<id> - id generated by MethodID.getMethodId(java.lang.String, java.lang.String, java.lang.String)
        • $LEVEL=<cond> - level match condition
